# Document Studio Editor

一个轻量级的在线文档编辑与素材整理页面，适合用来写方案、脚本、课程内容和简单图文稿件。

## 在线地址

GitHub Pages:

[https://corookie.github.io/document-studio-editor/](https://corookie.github.io/document-studio-editor/)

## 项目特点

- 支持基础富文本编辑，包括字体、字号、标题级别、加粗、斜体、下划线和项目符号
- 支持在文档中插入素材区域
- 素材区域支持上传图片和视频
- 图片显示缩略图，视频自动生成封面预览
- 素材支持删除和“另存为”
- 支持多文档与目录树结构展示
- 文档内容保存在浏览器本地，适合快速原型和轻量使用

## 当前界面能力

### 1. 文档编辑

页面提供一个简化版 Word 风格编辑区，可以快速编辑标题和正文内容。

### 2. 素材管理

点击“添加素材区域”后，可以在文档中插入素材块，并继续在素材块后面编辑正文内容。

### 3. 文件与目录

左侧提供目录树，可新建文件、新建目录、切换文档、重命名和删除。

## 本地使用

直接打开 `index.html` 即可运行。

如果你希望通过本地服务访问，也可以在项目目录下使用任意静态服务器启动。

## 文件结构

- `index.html`：页面结构
- `styles.css`：界面样式
- `script.js`：文档编辑、目录树和素材交互逻辑
- `.nojekyll`：GitHub Pages 静态站点兼容配置

## 当前限制

- 文档和目录数据保存在浏览器本地存储中，不是云端同步
- 素材文件本身不是长期云端存储，刷新或更换设备后不会像正式网盘系统那样持久保留
- 暂不支持多人协作编辑

## 后续可扩展方向

- 接入后端数据库，实现真正的远程保存
- 接入对象存储，持久化保存图片和视频
- 增加登录系统和权限控制
- 增加历史版本、自动保存和协作编辑能力
